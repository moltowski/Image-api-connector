### 🚀 Installation (5 minutes)

1. **Copier le dossier**:
   ```
   Copier: d:\custom node perso\ComfyUI-APIConnector\
   Vers: [Votre ComfyUI]\custom_nodes\ComfyUI-APIConnector\
   ```

2. **Redémarrer ComfyUI**

3. **Trouver le node**:
   - Clic droit → Add Node → image → api → **API Connector**

4. **Obtenir clé API**:
   - Aller sur https://fal.ai
   - Créer compte / Se connecter
   - Settings → API Keys → Create New
   - Copier la clé

5. **Premier test**:
   - Charger une image
   - Model: `nano_banana` (le moins cher)
   - Prompt: "Add vibrant colors"
   - Coller votre clé API
   - Exécuter!
